# Security Policy

## Supported Versions

This project is a student project, but security updates will be applied to the latest version only.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅ Yes     |
| Older versions | ❌ No      |

## Reporting a Vulnerability

If you discover a security issue in this project, please report it responsibly.

You can report security issues by:

* Opening a private issue on GitHub
* Or contacting the maintainer directly

## Contact

**Maintainer:** Aimash Waheed
GitHub: https://github.com/memonaimash

Please do not publicly disclose security vulnerabilities until they have been reviewed and fixed.

## Response Time

We aim to respond within a reasonable time and fix valid issues as soon as possible.

Thank you for helping keep this project safe.
