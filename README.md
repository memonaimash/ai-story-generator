# 📖 AI Story Generator

A Java Swing based desktop application that generates unique stories using customizable characters, genres, tones, lengths, and plot twists. This project demonstrates Object-Oriented Programming (OOP), GUI development, event handling, file management, and random story generation.

## ✨ Features

* 🎭 Custom Character Creation
* 📚 Automatic Character Backstories
* 🎲 Random Character Generation
* 🌍 Multiple Story Genres

  * Sci-Fi
  * Fantasy
  * Romance
  * Horror
  * Adventure
  * Realistic
* 🎨 Story Tone Selection

  * Epic
  * Dark
  * Light-hearted
* 📏 Adjustable Story Length

  * Short
  * Medium
  * Long
* 🔀 Random Plot Twists
* 💾 Save Stories as Text Files
* 🌙 Dark Mode Support
* 🖥️ Interactive Java Swing GUI

## 🛠️ Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* Event Handling
* File Handling
* Collections Framework
* Randomization Algorithms

## 🚀 How to Run

1. Clone or download the repository.
2. Open the project in NetBeans IDE.
3. Build and run the project.
4. Create characters and customize story settings.
5. Generate and save your unique story.
