# Contributing to Intelligent Story Generator

Thank you for your interest in contributing to the AI Story Generator project!

Contributions are welcome and appreciated. Whether you want to fix bugs, improve the user interface, add new story genres, or enhance functionality, your help is valuable.

## How to Contribute

### 1. Fork the Repository

Create your own copy of the repository by clicking the **Fork** button on GitHub.

### 2. Clone the Repository

```bash
git clone https://github.com/memonaimash/ai-story-generator.git
```

### 3. Create a New Branch

```bash
git checkout -b feature-name
```

### 4. Make Your Changes

Examples of contributions include:

* Adding new story genres
* Creating new plot twists
* Improving the GUI design
* Fixing bugs
* Optimizing code performance
* Improving documentation
* Enhancing dark mode support
* Adding new customization options

### 5. Commit Your Changes

```bash
git commit -m "Describe your changes"
```

### 6. Push Your Changes

```bash
git push origin feature-name
```

### 7. Submit a Pull Request

Open a Pull Request and provide a clear description of your changes and the problem they solve.

## Coding Guidelines

* Follow Java naming conventions.
* Write clean, readable, and maintainable code.
* Test your changes before submitting.
* Keep commits focused on a single feature or fix.
* Add comments where necessary.

## Reporting Issues

If you find a bug or have a feature request:

1. Open an Issue.
2. Provide a clear title and description.
3. Include steps to reproduce the issue if applicable.
4. Suggest possible improvements when possible.

## Suggested Future Enhancements

Contributors are encouraged to explore:

* AI-powered story generation
* PDF export functionality
* Story history management
* Character image support
* Text-to-speech integration
* Additional themes and UI improvements
* More genres and plot twists
* Database integration

## Code of Conduct

Please be respectful and professional when interacting with other contributors. Constructive feedback and collaboration help make this project better for everyone.
